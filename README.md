# App_BancoPy
Repositorio de una app simuladora de Cajero Automático – BancoPy

# INTRODUCCIÓN
BancoPy es una aplicación desarrollada para simular las funciones principales de un cajero automático, permitiendo al usuario interactuar con un sistema bancario básico de manera intuitiva y segura. Su propósito es replicar las operaciones más comunes que se realizan en un cajero real, como consultar el saldo, realizar depósitos y efectuar retiros, ofreciendo una experiencia simple, rápida y funcional. Este proyecto busca introducir al usuario en la lógica de la gestión financiera digital mediante una herramienta práctica basada en Python, promoviendo el aprendizaje de procesos bancarios y la correcta administración de recursos.

# OBJETIVO
El propósito de BancoPy es proporcionar una simulación clara y segura de las operaciones básicas de un cajero automático. La aplicación permite consultar el saldo en tiempo real, mostrando el monto disponible de forma inmediata; realizar depósitos de manera segura, actualizando el saldo automáticamente; y efectuar retiros controlados, autorizando la operación solo cuando existen fondos suficientes para evitar números negativos. Además, se caracteriza por ofrecer una interfaz sencilla con menús organizados, facilitando el uso para cualquier tipo de usuario. En conjunto, estas funcionalidades convierten a BancoPy en una herramienta educativa que enseña el funcionamiento esencial de un cajero automático dentro de un entorno práctico y accesible.

# CONTEXTO DEL PROBLEMA
En la vida cotidiana, muchas personas enfrentan dificultades para comprender cómo funcionan los cajeros automáticos o cómo se actualiza su saldo en función de los movimientos financieros que realizan. Esto puede generar confusiones, errores o inseguridad al momento de manejar dinero. Frente a esta realidad, BancoPy surge como una herramienta educativa que permite al usuario realizar operaciones bancarias simuladas sin riesgos, facilitando la comprensión de cómo cada acción, ya sea un depósito o un retiro, afecta su saldo disponible. De esta manera, BancoPy se convierte en una alternativa ideal para estudiantes y personas que desean aprender lógica de programación aplicada a servicios financieros, así como el uso básico y seguro de un cajero.

# ANÁLISIS REQUERIDO
BancoPy está estructurado para gestionar operaciones bancarias simples mediante un sistema interno que administra el saldo del usuario y procesa cada movimiento. La aplicación organiza sus funciones en un menú principal desde el cual se puede consultar el saldo, realizar depósitos o efectuar retiros. Cada operación es validada cuidadosamente: se impiden depósitos no válidos y se prohíben retiros que superen el saldo disponible. Los cambios se reflejan de inmediato, asegurando la integridad del monto actual. Esta estructura convierte a BancoPy en una herramienta eficiente para comprender el flujo de datos en una aplicación financiera, reforzando elementos como entradas de usuario, validaciones, actualización de variables y control de opciones.

# CARACTERÍSTICAS CLAVES DEL SITIO / APP
- Menú principal organizado con las opciones principales del sistema.
- Consulta de saldo en tiempo real.
- Función de depósitos con actualización inmediata del saldo.
- Retiros con validación de fondos suficientes.
- Mensajes claros de confirmación o error según la operación.
- Flujo sencillo que guía al usuario paso a paso.
- Lógica en Python ideal para fines educativos y de simulación bancaria.
